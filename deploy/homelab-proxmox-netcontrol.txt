Contenedor NetControl (panel WISP, este repo)
---------------------------------------------
- IP ejemplo en esta red: 192.168.13.73
- Panel web: http://<IP-CT>/netcontrol/
- Comprobar versión del panel: http://<IP-CT>/netcontrol/nc_version.php (esperado nc_panel_ui_rev=2)
- Ruta en el CT: /var/www/NetControl
- Instalar / actualizar sistema CT: deploy/lxc-bootstrap.sh
- Actualizar solo código (git ok): deploy/server-git-pull.sh
- Si git pull choca con cambios locales: deploy/reset-to-origin-main.sh (hace backup de config.php)
- Sin git: deploy/download-panel-from-github.sh

Actualizar código del CT sin SSH al .73
------------------------------
PVE 9.x no permite "ejecutar bash dentro del LXC" por la API HTTP. Opciones:
- Consola web: Nodo proxmox -> Shell -> pegar el comando que genera:
    deploy\pve-sync-netcontrol.ps1   (usa tu proxmox.credentials.json y encuentra vmid por IP)
- O SSH al hipervisor (.215) y el mismo "pct exec <vmid> -- bash -lc '...'"

Nota: Las IPs pueden cambiar; ajustá según tu red.
